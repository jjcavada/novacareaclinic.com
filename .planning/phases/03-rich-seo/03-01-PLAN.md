<task type="auto">
  <name>Task 1: Add Schema.org JSON-LD structured data</name>
  <files>src/app/layout.tsx</files>
  <action>
Add Schema.org JSON-LD structured data for MedicalClinic (which extends MedicalOrganization) in a script tag within the body.

Create the JSON-LD object with:
- @context: "https://schema.org"
- @type: "MedicalClinic"
- name: "NovaCare Clinic"
- description: Match the existing metadata description
- url: "https://novacareclinic.com"
- telephone: "(602) 555-0123" (from footer)
- address: { @type: "PostalAddress", streetAddress: "4523 W Glendale Ave, Suite 102", addressLocality: "Phoenix", addressRegion: "AZ", postalCode: "85301", addressCountry: "US" }
- geo: { @type: "GeoCoordinates", latitude: "33.5387", longitude: "-112.1350" } (Phoenix area)
- openingHours: "Mo-Fr 08:00-18:00"
- medicalSpecialty: ["Psychiatry", "Psychology", "MentalHealth"]
- availableService: Array of service types (Individual Therapy, Group Therapy, Psychiatric Evaluation, etc.)
- priceRange: "$$" (sliding fee scale mentioned)

Add via <script type="application/ld+json"> tag in the body, after the main content div. Use dangerouslySetInnerHTML with JSON.stringify for proper escaping.
  </action>
  <verify>npm run build passes; View page source shows JSON-LD script tag with valid JSON</verify>
  <done>MedicalClinic schema visible in page source with all required properties</done>
</task>
