# Milestone v1.0: SEO & Resources

**Status:** SHIPPED 2026-01-15
**Phases:** 1-4
**Total Plans:** 5

## Overview

Transform the existing NovaCare Clinic marketing website from placeholder content to a production-ready site with real authoritative behavioral health resources and comprehensive SEO optimization for search visibility.

## Phases

### Phase 1: Resources Content

**Goal**: Replace placeholder/mock PDF links with real, curated external resources from authoritative sources (NIMH, SAMHSA, NAMI, CDC) in both English and Spanish where official translations exist
**Depends on**: Nothing (first phase)
**Plans**: 1 plan

Plans:
- [x] 01-01: Real authoritative mental health resources

**Details:**
- Added 17 real mental health resources from authoritative sources
- Organized by condition: Depression, Anxiety, ADHD, Bipolar, PTSD
- Included Spanish translations from NIMH where available
- Added source attribution badges (NIMH, NAMI, CDC, SAMHSA)

### Phase 2: Core SEO

**Goal**: Implement foundational SEO elements across all pages for basic search engine discoverability
**Depends on**: Phase 1
**Plans**: 2 plans

Plans:
- [x] 02-01: Metadata and canonical URLs
- [x] 02-02: Heading hierarchy and internal linking

**Details:**
- Added metadataBase for site-wide canonical URL generation
- Implemented title template pattern across all pages
- Added SEO keywords to all 11 pages
- Fixed heading hierarchy issues
- Added 9 contextual internal links across 6 pages

### Phase 3: Rich SEO

**Goal**: Add structured data and social sharing metadata for enhanced search results and social previews
**Depends on**: Phase 2
**Plans**: 1 plan

Plans:
- [x] 03-01: Rich metadata (Schema.org, OpenGraph, Twitter Cards)

**Details:**
- Added Schema.org MedicalClinic JSON-LD structured data
- Added Open Graph meta tags for Facebook/LinkedIn sharing
- Added Twitter Card meta tags with large image format

### Phase 4: Technical SEO

**Goal**: Implement technical SEO requirements for crawlability and performance optimization
**Depends on**: Phase 3
**Plans**: 1 plan

Plans:
- [x] 04-01: XML sitemap and robots.txt

**Details:**
- Created XML sitemap with all 10 public pages
- Priority-based URL ordering (1.0 homepage to 0.3 legal)
- Created robots.txt allowing all crawlers with sitemap directive

## Milestone Summary

**Key Decisions:**
- External links only - Removed PDF downloads in favor of authoritative external sources
- Condition-based organization - Resources organized by mental health condition
- Source attribution - Added source badges to identify authority behind resources
- metadataBase pattern - Use Next.js metadataBase for canonical URL generation
- MedicalClinic schema - Use Schema.org MedicalClinic type for structured data
- Priority-based sitemap - homepage 1.0, services 0.9, main pages 0.8, resources 0.7, portal 0.6, legal 0.3

**Issues Resolved:**
- Replaced mock PDF resources with real authoritative links
- Fixed heading hierarchy across all pages
- Refactored legal-compliance-hub to support metadata export

**Issues Deferred:**
- Pre-existing TypeScript errors in SlidingFeeCalculator.tsx and PageTransition.tsx
- og-image.jpg creation (social platforms will show default until created)

**Technical Debt Incurred:**
- None
